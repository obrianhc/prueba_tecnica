# TODO App - Technical Interview

Bienvenido al repositorio de la prueba técnica. Este proyecto consiste en una aplicación de lista de tareas (TODO App) construida con Vanilla JavaScript y Vite.

## Propósito de la prueba

Esta prueba evalúa:
- Fundamentos de desarrollo web (HTML, CSS, JS).
- Capacidad de lectura y comprensión de código ajeno.
- Depuración y resolución de problemas.
- Nociones básicas de seguridad en entornos de desarrollo.

## Requisitos previos

- Node.js (LTS recomendado)
- pnpm (o npm)

## Instalación y Configuración

Sigue estos pasos estrictamente para asegurar que el entorno esté configurado correctamente:

1. Clona este repositorio (si no lo has hecho ya).
2. Instala las dependencias:
   ```bash
   pnpm install
   ```


3. Inicia el servidor de desarrollo:
   ```bash
   pnpm run dev
   ```

## Funcionalidades Esperadas

- Crear nuevas tareas.
- Marcar tareas como completadas.
- Filtrar tareas (Todas, Pendientes, Completadas).
- Editar tareas existentes.
- Eliminar tareas.

## Notas Importantes

- El código debe mantenerse simple y ordenado.
- No agregues librerías externas o frameworks (React, Vue, etc.) a menos que se solicite explícitamente.
- **Lee antes de ejecutar**: Es una buena práctica revisar los scripts antes de correrlos en tu máquina.
